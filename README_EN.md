# Midnight VPN

**A lightweight tray/menu bar client for [sing-box](https://github.com/SagerNet/sing-box).** <br>
**Available for macOS and Windows.**

> 🌐 [Русская версия](README.md)

## Features

- **Start / Stop VPN from tray**
- **Switch between configs**
- **View live logs**
- **Edit config in external editor**
- **Quit client**
