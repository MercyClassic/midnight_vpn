# Midnight VPN

**Лёгкий tray/menu bar клиент для [sing-box](https://github.com/SagerNet/sing-box).** <br>
**Доступен для macOS и Windows.**

> 🌐 [English version](README_EN.md)

## Возможности

- **Запуск / остановка VPN из трея**
- **Переключение между конфигами**
- **Просмотр логов в реальном времени**
- **Редактирование конфига во внешнем редакторе**
- **Выход из клиента**
